La aplicación consiste en un juego de preguntas y respuesta que se ejecutará mediante npm. El juego permite añadir preguntas
con sus respuestas, borrar preguntas, jugarlas de un modo aleatorio, probar una de ellas, listar las preguntas, y además
cuenta con un menú de ayuda. El acceso al juego se hará mediante un modelo cliente servidor, donde en el servidor se lanzará
el juego y en el cliente se pueden dar las órdenes que se quieran.
